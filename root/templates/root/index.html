Source: db-ip.com
Update: 00:00 UTC on the 3rd day of every month

You can get specific country's IP addresses to create dynamic routing rules for unblocking contents (like DMM or netflix), blacklist unwanted traffic by using this API
中文:
你可以使用這個API來取得特定國家的IP列表，制定特殊路由突破特定網站的封鎖(像是DMM或netflix等)，或是封鎖來自特定國家的連線。
日本語:
このAPIは特定国のIPアドレスリストを取得することができ、特殊ルーティングテーブルを設置しサイトの封鎖を突破するか(DMMとnetflixとか)、 特定国から来たトラフィックを遮断することもできます。
