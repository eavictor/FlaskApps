Request Example:

    
        http://www.example.com/geoip/api/routeos/ip_list/both/TW,JP

Response Example:

    
        /ip firewall address-list
        remove [/ip firewall address-list find list=AdServer]
        add address=1.160.0.0/12 list=AdServer
        add address=1.200.0.0/16 list=AdServer
        ...more...
        /ipv6 firewall address-list
        remove [/ipv6 firewall address-list find list=AdServer]
        add address=2001:238::/32 list=AdServer
        add address=2001:288::/32 list=AdServer
        ...more...

Example command to work in RouterOS

    
        /tool fetch url="http://www.example.com/ads/api/routeros/ip_list/both/ad_server_list"
        /import file-name=ad_server_list
        /file remove ad_server_list

Example Firewall Rules:

    
        /ip firewall filter
        add chain=forward action=reject reject-with=tcp-reset protocol=tcp dst-address-list=AdServer
        /ipv6 firewall filter
        add chain=forward action=reject reject-with=tcp-reset protocol=tcp dst-address-list=AdServer
